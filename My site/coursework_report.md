Подготовка среды разработки 

Для выполнения курсовой работы был установлен Python и генератор статических сайтов MkDocs. Установка выполнялась через менеджер пакетов pip. Затем была установлена тема оформления Material for MkDocs. В качестве среды разработки использовалась программа PyCharm.

Создание проекта MkDocs 

После установки инструментов был создан новый проект MkDocs с помощью команды `mkdocs new`. В результате была автоматически создана структура проекта с папкой `docs`, где размещаются страницы сайта, и файлом конфигурации `mkdocs.yml`.

Настройка конфигурации сайта 

В файле `mkdocs.yml` были заданы основные параметры сайта: название, описание, автор, тема оформления и структура навигации. Также была подключена тема Material для улучшения внешнего вида сайта.

Создание страниц сайта 

Были созданы основные страницы сайта: главная страница, страница «О себе», страница «Проекты» и страница «Контакты». Контент страниц создавался с использованием языка разметки Markdown. На страницах использовались заголовки, списки, таблицы, ссылки и изображения.

Добавление изображений и оформление сайта 

Для хранения изображений была создана папка `docs/images`. В неё были добавлены фотографии и изображения для оформления страниц. Также были добавлены кнопки со ссылками на социальные сети и выполнена базовая стилизация сайта с использованием возможностей темы Material.

Запуск и проверка сайта 

Для тестирования сайта был запущен локальный сервер с помощью команды `mkdocs serve`. После запуска сайт стал доступен в браузере по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000). Была проверена корректность отображения страниц, изображений и работы навигации
